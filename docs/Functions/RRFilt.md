# RRFilt Function

`RRFilt()` calculates how different a numeric vector is from its median, scaled by the standard deviation. Higher values indicate greater deviation.

## Usage
```R
RRFilt(x)
```

## Parameters
- `x`: Numeric vector.

## Returns
A single numeric score summarising the difference.
